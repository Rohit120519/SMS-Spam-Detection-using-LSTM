# SMS Spam Detection Using LSTM

## 📌 Overview
This project is a Django-based web application that detects whether an SMS message is **Spam** or **Ham (Not Spam)** using a Long Short-Term Memory (LSTM) deep learning model.

The application provides a simple user interface where users can enter an SMS message and receive an instant prediction.

---

## 🚀 Features

- Detects Spam and Ham messages
- LSTM-based Deep Learning Model
- Django Web Application
- Simple and User-Friendly Interface
- Real-time Prediction
- Trained on SMS Spam Collection Dataset

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Rohit120519/SMS-Spam-Detection-using-LSTM.git
```

Move into the project

```bash
cd SMS-Spam-Detection-using-LSTM
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Django server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
